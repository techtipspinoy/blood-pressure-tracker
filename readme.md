# Features

## User-Friendly Form
- Input fields for **SYS** (systolic), **DIA** (diastolic), and **PUL** (pulse)
- Input validation with appropriate ranges
- Clear visual indicators for units (**mmHg** and **/min**)

## Local Storage Database
- Uses the browser's `localStorage` as a file-based database
- Persists data between sessions
- Stores a timestamp with each measurement

## Records Display
- Shows the **10 most recent measurements**
- Displays **date and time** of each measurement
- **Color-coded status** based on blood pressure categories:
  - **Normal** → green
  - **Elevated** → yellow
  - **Hypertension Stage 1/2** → red

## Responsive Design
- Works seamlessly on **mobile**, **tablet**, and **desktop**
- Clean, modern interface with gradient accents

## Additional Functionality
- **Clear all records** button with confirmation dialog
- Save entries by pressing the **Enter key**
- Input validation with **helpful error messages**
